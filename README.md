# Planet Tours and Travelers Co. Ltd. Website

A complete, modern, and responsive travel and tourism website built for **Planet Tours and Travelers Co. Ltd.** based in Juba Town, South Sudan.

## 🌟 Features
- **Dynamic Tour Listings**: Filterable by destination and price.
- **Booking System**: Interactive booking form with WhatsApp integration.
- **Flight & Hotel Search**: Professional search interfaces for travel services.
- **Mobile Optimized**: Fully responsive design for all screen sizes.
- **Secure Payment Simulation**: Integrated payment gateway demo.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Modern design system with smooth transitions.
- **Vanilla JavaScript**: Dynamic content rendering and filtering logic.
- **Font Awesome**: Premium icons.
